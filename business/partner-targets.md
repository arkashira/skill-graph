# Partner Targets

## Integration Roadmap

### Phase 1: Immediate (0-3 months)
1. **GitHub**
   - **Rationale**: GitHub is the most widely used platform for code hosting and collaboration. Integrating with GitHub will allow users to visualize skill dependencies directly within their repositories.
   - **Free-Tier Limits**: 5000 API requests per hour.
   - **Integration Effort**: S
   - **Value-Add**: Solves the job of tracking and managing code dependencies for developers.

2. **GitLab**
   - **Rationale**: GitLab is another popular platform for code hosting and CI/CD. Integrating with GitLab will provide similar benefits to GitHub integration but for a different user base.
   - **Free-Tier Limits**: 6000 API requests per month.
   - **Integration Effort**: S
   - **Value-Add**: Solves the job of tracking and managing code dependencies for developers.

### Phase 2: Short-Term (3-6 months)
3. **Jira**
   - **Rationale**: Jira is widely used for project management and issue tracking. Integrating with Jira will allow users to link skill dependencies to specific project tasks and issues.
   - **Free-Tier Limits**: 10 users, 2GB storage.
   - **Integration Effort**: M
   - **Value-Add**: Solves the job of tracking skill dependencies in the context of project management.

4. **Slack**
   - **Rationale**: Slack is a popular communication tool for teams. Integrating with Slack will allow users to receive notifications and updates about skill dependencies directly in their Slack channels.
   - **Free-Tier Limits**: 10,000 messages per month.
   - **Integration Effort**: S
   - **Value-Add**: Solves the job of staying informed about skill dependencies without leaving the communication platform.

### Phase 3: Medium-Term (6-12 months)
5. **SonarQube**
   - **Rationale**: SonarQube is a widely used static code analysis tool. Integrating with SonarQube will allow users to visualize skill dependencies in the context of code quality and security.
   - **Free-Tier Limits**: Community edition is free.
   - **Integration Effort**: M
   - **Value-Add**: Solves the job of understanding skill dependencies in the context of code quality and security.

6. **Coverity**
   - **Rationale**: Coverity is another static code analysis tool. Integrating with Coverity will provide similar benefits to SonarQube integration but for a different user base.
   - **Free-Tier Limits**: Free trial available.
   - **Integration Effort**: M
   - **Value-Add**: Solves the job of understanding skill dependencies in the context of code quality and security.

### Phase 4: Long-Term (12+ months)
7. **Checkmarx**
   - **Rationale**: Checkmarx is a static code analysis tool focused on security. Integrating with Checkmarx will allow users to visualize skill dependencies in the context of code security.
   - **Free-Tier Limits**: Free trial available.
   - **Integration Effort**: L
   - **Value-Add**: Solves the job of understanding skill dependencies in the context of code security.

8. **GitHub CodeQL**
   - **Rationale**: GitHub CodeQL is a powerful static analysis tool. Integrating with GitHub CodeQL will allow users to visualize skill dependencies in the context of code security.
   - **Free-Tier Limits**: Included with GitHub Advanced Security.
   - **Integration Effort**: L
   - **Value-Add**: Solves the job of understanding skill dependencies in the context of code security.
