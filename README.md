# Skill Graph
A simple skill graph generator.

## Quick Start
1. Clone the repository: `git clone https://github.com/your-username/skill-graph.git`
2. Install dependencies: No dependencies required
3. Run the tests: `python -m pytest`
4. Run the API endpoint: `python -m skill_graph`
5. Use the API endpoint: `curl -X GET 'http://localhost:8000/generate-graph?target_skill=Python'`
